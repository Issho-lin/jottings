1、web前端三层
   结构层：HTML       从语义的角度描述页面结构
   样式层：CSS        从审美的角度美化页面
   行为层：JavaScript 从交互的角度提升用户体验

2、浏览器
   IE、Chrome、Firefox、Safari、Opera：内核（一般指渲染引擎）不一样

3、HTML标签（超文本标记语言）
   ⑴ 排版标签：h1-h6(标题)、p(段落)、br(换行)、hr(水平线)、div(独占一行)、span
   ⑵ 文本格式化标签：b/strong(加粗)、u/ins(下划线)、s/del(删除线)、i/em(斜体)【右侧标签语义强，利于搜索引擎抓取(SEO)】
   ⑶ img标签：src(指定图片路径)、alt(图片无法显示时显示的文字)、title(鼠标移到图片上显示的文字)
   ⑷ a标签：href(指定网址或锚点：外部网址+http://,锚点#id名)、target(指定打开链接方式：默认当前页面跳转值为_self,新页面跳转值为_blank)
   ⑸ 列表标签：ol(有序)、ul(无序)、dl(自定义：dt标题,dd内容)

   ⑹ table表格：caption(表格标题)、thead(表头)、tbody(表体)、tr(行)、td(单元格)、th(表头单元格)

   ⑺ form表单：action(设置提交的地址)、method(提交方式get或post)
      ♦属性：value(设置文本框或者密码框的默认值或者按钮的显示文字)、disabled(禁用某个元素)、placeholder(占位符)、checked(单选或多选默认选中)、                                                    selected（option默认选中)
      ①input标签：type属性值不同就是不同的元素（text文本框、password密码框、radio单选框、checkbox多选框、file上传标签、button普通按钮、submit提交按                        钮、reset重置按钮、image图片按钮）

       ③文本域textarea：大段文本
   ⑻ audio和video标签：src(指定资源路径)、autoplay(自动播放)、controls(显示播放控件)、loop(循环播放)
      ♦兼容：source指定不同格式的资源路径

4、CSS三种写法
   内联：写在head标签里，写一个style标签，style标签写css代码
   外联：写在.css文件里面，然后用link标签来导入，href属性
        行内：给标签加一个style属性，属性里面写CSS代码

5、CSS选择器
   ⑴ 标签选择器、类名选择器(.class)、id选择器(#id)
   ⑵ 通配符*：代表匹配页面上的所有标签，但是效率低，不推荐使用
   ⑶ 后代选择器：选择器（空格）选择器，代表找到左边选择器里面所有的匹配的后代
   ⑷ 子代选择器：选择器>选择器，代表找到左边选择器里的子元素里匹配的元素
   ⑸ a标签的伪类选择器：a:link(未点击前)、a:visited(点击后)、a:hover(鼠标悬停)、a:active(鼠标点击)
   ⑹ 位置伪类选择器：li:first-child(既要满足是第一个子元素，还要满足是li标签)、li:last-child(既要满足是最后一个子元素，还要满足是li标签)、li:nth-child(n)(n就        是数字，n为几就代表找到第几个子元素，并且是li标签,n也可以是odd （奇数）和 even（偶数）,4n 代表4的倍数)
   ⑺ 交集选择器：如.box.ok,代表既要有box这个类，还要有ok这个类，要连在一起写，中间不能有空格或逗号等
   ⑻ 并集选择器：选择器,选择器，就是把一堆选择器并在一起用同一个样式，一般是用来替代通配符的，用来设置一些标签的默认样式
       ⑼ 伪元素选择器：::first-letter(首字母)、::first-line(首行)、::selection(选中行)

6、CSS属性
   ⑴ 文字(font)：color[文字颜色：单词、16进制#ffffff、rgb(0,0,0)]、font-size(字体大小)、font-weight(文字粗细)、font-style(是否斜体)、font-family(字体库)
      ♦font连写:style weight size family
   ⑵ 背景(background)：backround-color(背景颜色)、background-img(背景图片)、background-size(背景图片大小：先写宽再写高,按比例缩放:cover一边到了就                        停,contain两边到盒子才停)、background-repeat(是否平铺：repeat、repeat-x、repeat-y、no-repeat)、background-position(背景图片位置：100px 50px向右                    100像素向下50像素文字写法left,right,top,bottom,center、百分比写法50% 50%)、background-attachment(背景图片是否随着滚动而滚动,scroll滚动、fixed固定)
      ♦background连写：路径 平铺 位置 大小 颜色（可以只写背景图片，也可以再加一个是否平铺，代表左上对齐，图片本身的大小，也可以只写背景颜色，多张背景                        图 片用逗号隔开）
        ⑶ 文字对齐：text-align(left、center、right)、line-height(设置跟盒子高度一样，就垂直居中了，如果没设置高度，它也是垂直居中，只不过盒子的高度就是你设置的行        高的高度)、text-decoration(none、下划线underline、删除线line-through、上划线overline)
        ⑷ 透明度：opacity：0-1(整个元素的透明度)、background:rgba()[四个参数分别是，红，绿，蓝，透明度]
        ⑸ 边框(border)：border:1px solid black(边框的样式:实线solid、虚线dashed、点线dotted、双实线double)
        ⑹ 内边距(padding)：设置内容和边框的距离，默认四个方向顺序：上、右、下、左，它会撑开盒子的宽度（相当于加了一部分）
        ⑺ 外边距(margin)：设置的是盒子与外部的距离，默认四个方向顺序：上、右、下、左，某些情况下会发生塌陷现象
      ♦共享塌陷(两个盒子如果在同一行，那么设置其中一个的margin-top，同一行的所有元素都会跟着走，解决：避免使用)
      ♦垂直塌陷(如果上下两个盒子，上面的盒子设置了下边距，下面的盒子设置了上边距，那么会取最大值作为之间的间距)
      ♦包含塌陷(给子盒子设置margin-top，影响的是父盒子，解决：给父盒子加上边框，或者写 overflow:hidden（推荐）)
        ⑻ 首行缩进(text-indent:4em)：代表首行缩4个汉字
        ⑼ 合并单元格边框(border-collapse:collapse)：给table加的样式
        ⑽ 对齐方式(vertical-align)：baseline(默认基线对齐)、middle(中线对齐)

7、CSS的三大特性
        ⑴ 继承：父元素的部分样式可以继承给子元素(a标签不能继承颜色、h系列标签不能继承文字大小、背景图片和背景颜色，宽和高特殊情况被继承)
    ⑵ 层叠：权重一样的情况下，相同属性的话，后面的覆盖前面的，不同属性合在一起都很用
    ⑶ 优先级：选择器的权重，!important > 行内 > id > 类 > 标签 > 通配符 > 继承，如果不同级别，级别高的权重最大，如果级别相同，就看谁的数量多

8、元素
        ⑴ 块级元素：独占一行的元素，可以设置宽和高，例：div，p，h系列，ul,li,ol
    ⑵ 行内元素：不会独占一行，不能设置宽和高，例：a,b,u，span等等
      ♦行内元素的特殊性：不能使用垂直外边距，但是可以水平外边距，内间距四个方向都可以
    ⑶ 行内块元素：不会独占一行，能设置宽和高，例：img
      ♦元素转换：display    [none(隐藏消失)、inline(变成行内元素)、inline-block(变为行内块元素)、block(变为块级元素)]

9、盒子模型
⑴盒子真实宽高=设置宽/高 + 左右内边距/上下内边距 + 左右边框/上下边框
⑵ 外扩和内减(box-sizing)：border-box(内减)、content-box(外扩，默认值)

10、浮动
        ⑴ float：left(左浮动)、right(右浮动)
        ⑵ 细节：
      ♦当一个元素浮动后，会脱离标准流：不占原来的位置，后面的元素会把它的位置顶上
      ♦浮动会改变元素的显示模式：会把行内元素变为（类似）行内块元素
      ♦当多个元素浮动，按他们排列的顺序，依次从左至右浮起来，浮动后它们是顶部对齐
      ♦当元素浮动，它里面的子元素也会跟着一起动
      ♦浮动的范围只是在它父元素里面
      ♦当父元素不够放浮动元素时，会另起一行
      ♦浮动的阻断（卡住）现象：在浮动的过程中，如果一行不够放，要另起一行，但是在另一行的时候要看中间有没突出阻断的，如果有，就放在阻断的后面（阻断的                        后面还是不够放就另起一行）
      ♦元素在浮动的时候会找位置：找它上一个元素是不是浮动，如果是浮动，就接在它后面浮动，如果没有，就一直浮动到父元素的边界上（浮动方向）
        ⑶ 清除浮动(clear)：left(不让自己左边有浮动元素)、right(不让自己右边有浮动元素)、both(不让自己左右有浮动元素)
      ♦这个属性不能清除任何元素的浮动属性，也只能影响自己，不影响别的元素。
        ⑷ 解决浮动带来的影响：
      ♦方法一：直接给父元素高度
      ♦方法二：overflow:hidden      
      ♦方法三：标签法，给父元素的最后加一个块级标签，设置块级标签的样式clear:both
      ♦方法四：伪元素法
  1. /*解决浮动带来的影响，最常用固定写法*/
  2. .clearfix::after{
  3.     content:'';
  4.     display:block;
  5.     clear:both;
  6.     height:0;
  7.     visibility:hidden;  
  8. }
  9. /*解决兼容性问题*/
  10. .clearfix{
  11.     *zoom:1;
  12. }

11、定位
⑴ 定位模式
      ♦relative：相对定位，相对于其原文档流的位置进行定位
      ♦absolute：绝对定位，相对于其上一个已经定位的父元素进行定位
      ♦fixed：固定定位，相对于浏览器窗口进行定位
      ♦static：静态定位（默认定位方式）所有的标签在一开始都是静态定位（标准流）
